# RedRockFE-cli

## 开发测试指南

输入命令进入开发模式

```shell
pnpm start
pnpm link --global
```

脚手架启动

```shell
rrfe create [project]
```

## Done

包括但不仅限于以下功能：

- 使用 Vite 作为打包工具(未来可能有变化)
- 一键配置 tailwind，typescript，eslint，prettier
- 初始化 git 以及配置 husky
- 初始模板(目前只有 React)

> 以上内容均可以自己修改
